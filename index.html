Onboarding Curriculum

เรียน Vue.js + Vuex + Kendo UI เพื่อทำงานกับ KL.Core.UI

บทเรียนชุดนี้ทำขึ้นสำหรับคนที่ ไม่เคยเขียน Vue.js / Vuex / Kendo UI มาก่อนเลย โดยจะเริ่มจากพื้นฐานล้วน ๆ (ที่ใช้ได้กับโปรเจกต์ Vue ทั่วไปในโลก ไม่ใช่แค่โปรเจกต์นี้) แล้วค่อย ๆ โยงเข้าสู่วิธีที่โปรเจกต์ KL.Core.UI (Faco Core) เลือกใช้งานจริง รวมถึงพาไปรู้จัก Shared Component และ Router/Store ที่ทีมสร้างไว้ให้แล้ว จะได้ไม่ต้องเขียนซ้ำ

Stack ที่โปรเจกต์นี้ใช้จริง (สรุปสั้น ๆ)
  • Vue 2.7.14 (Options API) — ไม่ใช่ Vue 3
  • Vuex 3.1.1 ร่วมกับ vuex-pathify (helper ที่ทำให้เขียน state สั้นลง)
  • Vue Router 3 (history mode) — และ route ถูก "auto สร้าง" จากโครงสร้างโฟลเดอร์ views เอง
  • Kendo UI for Vue (@progress/kendo-vue-*, kendo-*-vue-wrapper) เป็น UI component หลัก (Grid, DropDown, DatePicker, Window, TreeView ฯลฯ)
  • Bootstrap-Vue สำหรับ layout พื้นฐาน (b-row, b-col, b-container)
  • Vuelidate สำหรับ validate ฟอร์ม, vue-i18n สำหรับหลายภาษา (TH/EN), axios สำหรับเรียก API, SignalR สำหรับ real-time
  • Build ด้วย Webpack (ไม่ใช่ Vite) แบ่งเป็นหลาย module (Finance, Collection, Legal, HPAdmin ฯลฯ) ที่ build แยกกันได้ผ่าน SCOPE_MODULE
